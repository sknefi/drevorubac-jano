# Jano Drevorubač
Dvojdňový python grafický projekt v pygames

Sprite
+ `Skeleton` - Prechádza sa po mape
+ `Drevorubac` - Jeho úlohou je získať čo najväčší dmg údermi do skeletonov
+ `Drevorubac úder` - Drevorubačova abilita udrieť
